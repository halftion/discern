# discern

#### 介绍
“谛听”（discern）资产识别分析平台，IOT—Scanner的迭代优化版本，使用爬虫创建数据库，无需数据集即可使用

#### 主要架构
1. 前端 react
2. 服务端 flask
3. 爬虫 scrapy
4. 数据库 mysql

#### 代码目录

1.  Server - flask服务端
2.  Spider - 数据爬虫
3.  Client - react前端
